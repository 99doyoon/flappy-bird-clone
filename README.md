# Flappy Bird Clone

Unity를 공부하면서 만든 간단한 Flappy Bird 스타일 게임입니다.
작은 기능들을 하나씩 구현하면서 Unity의 기본적인 구조와 기능을 연습하는 것을 목표로 했습니다.

## 학습한 내용

이 프로젝트를 만들면서 다음과 같은 것들을 직접 구현해보았습니다.

* Unity 2D 프로젝트 구조 이해
* Rigidbody2D를 이용한 플레이어 점프 구현
* Collider를 이용한 충돌 처리
* Prefab을 이용한 오브젝트 생성
* Spawner를 이용한 파이프 반복 생성
* Trigger를 이용한 점수 시스템 구현
* GameManager를 이용한 게임 상태 관리
* Scene을 분리하여 메뉴 화면과 게임 화면 구성
* TextMeshPro를 이용한 UI 점수 표시
* PlayerPrefs를 이용한 High Score 저장
* 간단한 배경 스크롤 구현

## 조작 방법

PC

* Space : 점프

Mobile

* 화면 터치 : 점프

## 실행 방법

Unity에서 프로젝트를 열고 `MenuScene`을 실행하면 게임을 플레이할 수 있습니다.

## 사용 기술

* Unity
* C#
